Introduction to FootyHUB's web application

This assignment explores sports data using the API on Squiggle. This is a single page application that will allow the user to explore the performance of their team with respect to other teams in the league. The application incorporates user stories provided by our professor and permits the user to mark a selected team as their “favourite”

Note: AFL data that is presented belongs to the 2019 season starting on round 20.
